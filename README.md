# AR-VR-Japan-Knick-Knack - Huy Nguyen

An AR application made with Unity that displays a small interactive 3D scene representing Kyoto, Japan. The application visualizes real-time weather information and time-based environmental changes while allowing users to interact with a 3D character.

# Features
- Real-time temperature data from OpenWeather API
- Weather-based visual effects (sun, rain, clouds)
- Time-based scene
- Text display using TextMeshPro
